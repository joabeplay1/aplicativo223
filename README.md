# Omega Factory ⚡

Aplicativo pronto para produção — deploy direto na Vercel, **sem instalar Node.js e sem rodar npm install**.

## Deploy em 3 passos

1. Envie o conteúdo deste ZIP para um repositório no GitHub (arraste os arquivos soltos, não a pasta ZIP).
2. Acesse https://vercel.com → **Add New Project** → importe o repositório.
3. A Vercel detecta o site estático automaticamente → clique em **Deploy**. Pronto! 🚀

## Estrutura do projeto

- `index.html` — página principal
- `css/style.css` — estilos
- `js/script.js` — lógica do app
- `assets/favicon.svg` — ícone
- `vercel.json` — configuração de deploy (site estático, sem build)
- `.gitignore`

## Rodar localmente (opcional)

Basta abrir `index.html` no navegador. Não há dependências para instalar.
